This quickstart guide will walk you through the process of setting up and using Parley in your site.

After this guide you'll know how to add Parley into another project.

<Warning>
  Before starting this guide, make sure to create an [account on Parley](https://parleyai.net/dashboard) and created your first product
</Warning>

## Getting Started

After you created an account and have created your product, you need to get the <u>product token from you dashboard.</u>

<u>The format of token is like this</u>

```
eyJhbGciOiJIUzI1NiJ9.eyJwcm9kdWN0SWQiOiIzOGY1MzViNi0xNmJjLTRmMzItODliY.....
```
